# Sales Data Analysis using Python

## Project Overview
This project demonstrates basic data analysis on a sales dataset using Python.  
The goal is to clean the data, explore key patterns, and extract useful insights.

## Dataset
The dataset contains sample sales records including:
- Product
- Category
- Region
- Sales
- Profit

## Tools Used
- Python
- Pandas
- CSV dataset

## Key Analysis
- Total sales by region
- Profit analysis
- Category performance
